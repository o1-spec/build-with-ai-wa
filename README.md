# AgriMind Voice 🌿

**AgriMind Voice** is a multilingual, voice-first AI assistant specifically designed for smallholder farmers in Nigeria. It provides instant crop diagnostics and actionable agricultural advice in local languages, helping farmers make better decisions directly in the field.

## 🌟 Features

- **🗣️ Voice-First Interaction**: Designed for farmers who prefer speaking over typing. Integrated mock voice-to-text for seamless reporting.
- **🌍 Multilingual Support**: Full support for **English**, **Yoruba**, **Hausa**, and **Igbo**, ensuring accessibility across diverse regions.
- **🔍 Instant AI Diagnosis**: Analyzes crop symptoms and provides a likely diagnosis of pests or diseases.
- **📋 Actionable Plans**: Delivers immediate treatment steps and long-term prevention strategies.
- **⚠️ Risk Assessment**: Categorizes issues by risk level (Low, Medium, High) to help farmers prioritize their actions.
- **🎨 Organic UI**: A warm, nature-inspired interface designed for clarity and ease of use under field conditions.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/) with [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/agrimind-voice.git
   cd agrimind-voice
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000`.

## 📖 How to Use

1. **Select Language**: Choose your preferred language from the globe icon in the top right.
2. **Enter Details**: Select your crop type and enter your location.
3. **Describe the Issue**: Either type your observation or use the **Voice Input** button to describe what's happening to your crops.
4. **Submit**: Click "Submit Report" to receive an AI-powered analysis.
5. **Review Advice**: Follow the "Immediate Action" and "Prevention" steps provided in the response.
